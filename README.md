## :rocket: Sobre o nível

Nesse nível foi criada uma aplicação voltada para criar transações financeiras do usuário, alguns doc conceitos vistos:

- Miragejs (Criaçaõ de um fake backend, incluindo seed);
- Modal (utilização da library react-modal);
- Formatação de dados;
- Componentização;
- Estado e Imutabilidade;
- Hooks React;
- Uso do context API;
- Criação de formuário.

### Tech Stack

- ReactJS
- Typescript
- MirageJS
